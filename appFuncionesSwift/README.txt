----------------------------------------------------
1. Overview
----------------------------------------------------
appFuncionesSwift es una aplicación de consola desarrollada en Swift cuyo propósito es
demostrar el uso de funciones con parámetros, retorno de valores, tuplas, recursividad,
tipos numéricos (Double, Int) y generación de números aleatorios.

El programa contiene múltiples funciones que operan sobre arreglos y números ingresados
(o generados aleatoriamente), incluyendo cálculos matemáticos comunes como factorial 
iterativo y recursivo, multiplicación y búsqueda del valor mayor y menor dentro de un
conjunto de números.

Este proyecto introduce principios fundamentales de desarrollo en Swift tales como:
- Manipulación de arrays.
- Funciones con distintos tipos de parámetros.
- Retorno múltiple mediante tuplas.
- Entrada del usuario desde consola.
- Generación de valores aleatorios con Int.random(in:).

----------------------------------------------------
2. Flujo General del Programa
----------------------------------------------------
El programa ejecuta las siguientes acciones en secuencia:

1. Muestra un arreglo estático y determina su valor mayor y menor.
2. Solicita dos valores al usuario (o los genera aleatoriamente) y aplica la función 
   de multiplicación.
3. Solicita un número (o lo genera aleatoriamente) para calcular su factorial de forma:
   - Iterativa
   - Recursiva
4. Genera un número aleatorio de dos dígitos y lo muestra en consola.

----------------------------------------------------
3. Conceptos de Swift Aplicados
----------------------------------------------------
• Funciones con parámetros y valores retornados.  
• Tuplas como mecanismo para devolver múltiples datos.  
• Uso de estructuras de control: ciclos y condicionales.  
• Entrada y salida estándar mediante `readLine()` y `print()`.  
• Conversión y validación de tipos (`Int`, `Double`).  
• Funciones recursivas y manejo de límites para prevenir overflow.
